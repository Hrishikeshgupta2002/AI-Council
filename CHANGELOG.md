# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of Agentic Council System
- Beautiful CLI interface with rich formatting
- 4 specialized agents (Elon, Sam, Sheryl, Ray)
- Natural group chat conversations
- Tag agents with @mentions for focused debates
- Weighted decision model
- Synthesis agent for comprehensive analysis
- Configuration management via environment variables
- Support for multiple input methods (CLI args, piped input, interactive)

### Features
- Parallel agent processing
- Real-time response display
- Agents can choose to respond or skip
- 2-3 exchange debates for tagged agents
- Structured output with agreements, conflicts, and blind spots
